# 반응형 회원가입 페이지
## 1. 개요
- 뉴모피즘(Neumorphism) 디자인을 적용한 회원가입 페이지입니다.
- 사용 스택 : HTML, CSS, JavaScript, LocalStorage


## 2. 기능
#### 2-1. 반응형 웹

#### 2-2. 유효성 검사
- 제출 버튼을 클릭하면 입력된 정보를 모두 확인하여 Pass(초록)와 Non-Pass(빨강)를 보여줍니다.
- 모두 Pass면 toast를 띄워 가입에 성공했음을 알립니다.
